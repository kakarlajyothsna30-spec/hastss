# 🚦 SmartFlow — Adaptive Traffic Management System

A modern, interactive web application that demonstrates an **AI-powered adaptive traffic management system** designed to replace static signal timers with real-time, human-aware traffic logic.

---

## 🌟 Features

- **Hero Section** — Overview and mission statement of SmartFlow
- **Challenges** — Visualizes real-world problems with traditional traffic systems
- **Problem & Solution** — Side-by-side comparison of legacy vs. adaptive approaches
- **System Workflow** — Step-by-step flow of how the AI system operates
- **System Architecture** — Component breakdown and tech stack diagram
- **System Logic** — Core decision-making algorithms explained
- **Traffic Simulation** — Interactive live simulation of an adaptive intersection
- **Real-Time Dashboard** — Live metrics for signal timing, vehicle throughput & more
- **Traffic Tester** — Input custom scenarios and observe AI response

## 🚀 Getting Started

### Prerequisites
- Node.js **v18+**
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/kakarlajyothsna30-spec/hastss.git
cd hastss

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```
